# Счет-оферта для 1С:Бухгалтерии 3.0

Демо-проект [Конвейера 1С](https://gitsell.ru/1c) от Gitsell: расширение для **1С:Бухгалтерии предприятия 3.0**, которое добавляет отдельную печатную форму **"Счет-оферта"** к документу **"Счет на оплату покупателю"**.

Проект показывает, как из небольшой продуктовой идеи можно собрать рабочее расширение 1С: с заимствованием типового объекта, MXL-макетом, серверной логикой печати, проверкой в демо-базе и публичным репозиторием.

## Что Демонстрирует Конвейер 1С

- Быстрый выпуск прикладной доработки 1С в формате расширения, без изменения основной конфигурации.
- Аккуратное подключение к типовому механизму печати БП через `&После("ПриПолученииКомандПечати")`.
- Отдельный идентификатор печати `ГСК5664_СчетОферта`: типовой счет не подменяется.
- MXL-макет, который можно редактировать штатно через **Макеты печатных форм**.
- Серверную проверку результата через HTTP bridge без ручного кликанья в интерфейсе.
- Опциональный брендинг: `Создано на Конвейере 1С - https://gitsell.ru/1c`.

## Пользовательский Сценарий

В документе **"Счет на оплату покупателю"** в меню **Печать** появляется команда:

```text
Счет-оферта (Конвейер 1С)
```

Печатная форма формирует обычный счет на оплату, но добавляет юридический смысл оферты:

- заголовок `СЧЕТ-ОФЕРТА`;
- текст основания: счет является офертой по ст. 435 ГК РФ;
- условие акцепта: полная оплата по реквизитам счета по ст. 438 ГК РФ;
- табличную часть товаров, суммы, НДС и итог к оплате;
- подписи руководителя и бухгалтера;
- опциональную подпись сервиса Gitsell в подвале.

Печатная форма проверена в демо-базе `fresh-bp-demo` через `CodexTestBridge` 9 июля 2026 года.

Тестовый документ:


## Состав Расширения

| Объект | Тип | Назначение |
|--------|-----|------------|
| `ГСК5664_ПечатьСчетОферта` | Обработка | Менеджер печати и сборка табличного документа |
| `СчетНаОплатуПокупателю.ПФ_MXL_СчетОферта` | Макет | MXL-макет печатной формы |
| `ГСК5664_ОбщегоНазначения` | Общий модуль | Служебные функции, URL сервиса, текст брендинга |
| `ГСК5664_ПоказыватьБрендинг` | Константа | Включает подпись Gitsell в печатной форме |
| `УправлениеПечатьюПереопределяемый` | Заимствованный общий модуль | Добавляет команду печати через расширение |
| `СчетНаОплатуПокупателю` | Заимствованный документ | Контейнер для MXL-макета |

Анализ расширения:

```text
[OWN]      Role.ГСК5664_ОсновнаяРоль
[OWN]      CommonModule.ГСК5664_ОбщегоНазначения
[BORROWED] CommonModule.УправлениеПечатьюПереопределяемый
[OWN]      Constant.ГСК5664_ПоказыватьБрендинг
[BORROWED] Document.СчетНаОплатуПокупателю
[OWN]      DataProcessor.ГСК5664_ПечатьСчетОферта
```
