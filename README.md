# Счет-оферта / счет-договор для БП (1Фреш #5664)

Расширение для **1С:Бухгалтерия предприятия 3.0** (в т.ч. демо в облаке 1Фреш): отдельная **печатная форма «Счет-оферта»** для разовых продаж, когда нужен счет с текстом оферты без отдельного договора.

**Сделано на [Конвейере 1С](https://gitsell.ru/1c)** (Gitsell).

- **Идея сообщества:** [Центр идей 1Фреш #5664](https://dev.1cfresh.com) — счет-оферта для разовых продаж.
- **Версия расширения:** 0.1.0.0  
- **ИБ для проверки:** `fresh-bp-demo` (кластер) + веб `http://192.168.2.127/fresh-bp-demo`.

## Что делает расширение

1. В документе **«Счет на оплату покупателю»** в меню **Печать** появляется команда **«Счет-оферта (Конвейер 1С)»**.
2. Печать выполняется через обработку `ГСК5664_ПечатьСчетОферта`; макет **`ПФ_MXL_СчетОферта`** на документе `СчетНаОплатуПокупателю` — редактируется в **Макеты печатных форм** (генерация: Layout MCP, см. [docs/LAYOUT-MCP.md](../docs/LAYOUT-MCP.md)).
3. Форма не подменяет типовой счет на оплату — это **отдельный** идентификатор печати `ГСК5664_СчетОферта`.
4. Опциональный брендинг в подвале (константа `ГСК5664_ПоказыватьБрендинг`, общий модуль `ГСК5664_ОбщегоНазначения`).

## Состав метаданных

| Объект | Назначение |
|--------|------------|
| `ГСК5664_ПечатьСчетОферта` | Обработка печати |
| `СчетНаОплатуПокупателю.ПФ_MXL_СчетОферта` | MXL-макет (пользовательское редактирование) |
| `ГСК5664_ОбщегоНазначения` | Регистрация команд печати, брендинг |
| `ГСК5664_ПоказыватьБрендинг` | Вкл/выкл подпись «Создано на Конвейере 1С» |
