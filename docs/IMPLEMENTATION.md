# План реализации

## MVP для аудита 1Фреш

1. Макет `ГСК5664_СчетОферта` (MXL): реквизиты счета + текст оферты (шаблон в настройках).
2. Регистрация в подсистеме печати документа-источника.
3. Опциональное включение в настройках расширения (константа «Использовать счет-оферту» — добавить при необходимости).
4. Карточка магазина: «Создано на Конвейере 1С», ссылка https://gitsell.ru/1c.

## Ограничения

- Не дублировать типовой счет на оплату — отдельное имя макета.
- Юридический текст оферты согласовать с заказчиком; брендинг только в футере.

## Проверка

- Печать из документа, PDF, подпись ЭДО (если применимо).
- БП в 1Фреш, версия совместимости = базовой.
