# Техническое Описание

Инженерные детали расширения **"Счет-оферта"** для 1С:Бухгалтерии предприятия 3.0.

## Проверка Через Bridge

Печатная форма проверена в демо-базе `fresh-bp-demo` через `CodexTestBridge` 9 июля 2026 года.

Тестовый документ:

```text
Счет покупателю 0000-000001 от 29.08.2025 12:00:00
```

Результат серверного рендера:

| Проверка | Результат |
|----------|-----------|
| Табличный документ сформирован | OK |
| Размер табличного документа | 48 строк x 23 колонки |
| Заголовок `СЧЕТ-ОФЕРТА` найден | OK |
| Текст оферты найден | OK |
| Условие акцепта найдено | OK |
| Итоги и сумма к оплате найдены | OK |
| Брендинг | Выключен константой в демо-базе |

Фрагмент текстовой выгрузки:

```text
СЧЕТ-ОФЕРТА № 1 от 29.08.2025

Оферта: Настоящий счет является офертой (ст. 435 ГК РФ);
акцепт - полная оплата по реквизитам счета (ст. 438 ГК РФ).

Всего к оплате: 10 400,00
Настоящий документ является офертой ...
Акцепт оферты - полная оплата по реквизитам ...
```

Команда проверки доступности bridge:

```powershell
python C:\Users\Манжела Александр\.codex\skills\codex-test-bridge\client.py `
  --base-url http://192.168.2.127/fresh-bp-demo/hs/codex-test health
```

Основной рендер выполнялся серверным вызовом:

```bsl
ТабличныйДокумент =
    Обработки.ГСК5664_ПечатьСчетОферта.СформироватьПечатнуюФорму(
        МассивОбъектов,
        Неопределено);
```

## Состав Расширения

| Объект | Тип | Назначение |
|--------|-----|------------|
| `ГСК5664_ПечатьСчетОферта` | Обработка | Менеджер печати и сборка табличного документа |
| `СчетНаОплатуПокупателю.ПФ_MXL_СчетОферта` | Макет | MXL-макет печатной формы |
| `ГСК5664_ОбщегоНазначения` | Общий модуль | Служебные функции, URL сервиса, текст брендинга |
| `ГСК5664_ПоказыватьБрендинг` | Константа | Включает подпись Gitsell в печатной форме |
| `УправлениеПечатьюПереопределяемый` | Заимствованный общий модуль | Добавляет команду печати через расширение |
| `СчетНаОплатуПокупателю` | Заимствованный документ | Контейнер для MXL-макета |

Анализ расширения:

```text
[OWN]      Role.ГСК5664_ОсновнаяРоль
[OWN]      CommonModule.ГСК5664_ОбщегоНазначения
[BORROWED] CommonModule.УправлениеПечатьюПереопределяемый
[OWN]      Constant.ГСК5664_ПоказыватьБрендинг
[BORROWED] Document.СчетНаОплатуПокупателю
[OWN]      DataProcessor.ГСК5664_ПечатьСчетОферта
```

## Точки Подключения

Команда печати добавляется через заимствованный общий модуль `УправлениеПечатьюПереопределяемый`:

```bsl
&НаСервере
&После("ПриПолученииКомандПечати")
Процедура ГСК5664_ПриПолученииКомандПечати(Знач ПолноеИмяОбъектаМетаданных, КомандыПечати)
```

Команда регистрирует:

```bsl
КомандаПечати.МенеджерПечати = "Обработка.ГСК5664_ПечатьСчетОферта";
КомандаПечати.Идентификатор = "ГСК5664_СчетОферта";
КомандаПечати.Представление = НСтр("ru = 'Счет-оферта (Конвейер 1С)'");
```

Печатная форма формируется в менеджере обработки `ГСК5664_ПечатьСчетОферта`.
