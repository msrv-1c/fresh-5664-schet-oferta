# Примеры печатной формы

Файлы сформированы из демо-документа `Счет покупателю 0000-000001 от 29.08.2025 12:00:00`.

- `schet-oferta.html` - HTML-выгрузка печатной формы.
- `schet-oferta.png` - PNG-превью HTML-выгрузки.
- `schet-oferta.xlsx` - Excel-версия печатной формы.
- `schet-oferta.txt` - текстовая выгрузка для быстрой проверки содержимого.
