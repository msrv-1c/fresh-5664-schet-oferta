# Заимствование (БП 3.0)

После подключения выгрузки базовой конфигурации:

```text
/cfe-borrow <ConfigPath> <ExtensionSrc> Document.СчетНаОплатуПокупателю
```

Макет печати `ПФ_MXL_СчетОферта` добавлен в заимствованный документ (редактирование в «Макеты печатных форм»).

Если печать вешается на реализацию:

```text
/cfe-borrow ... Document.РеализацияТоваровУслуг
```

Команды печати для заимствованного документа **нельзя** вешать на `ManagerModule` документа в расширении
(ibcmd: «ManagerModule не является подчинённым для объекта Document»).

Рабочий перехват — заимствование `CommonModule.УправлениеПечатьюПереопределяемый` и
`&После("ПриПолученииКомандПечати")` (см. `src/CommonModules/УправлениеПечатьюПереопределяемый/Ext/Module.bsl`).
